# 航海家號：太陽系引力彈弓任務

這是一個以單一 HTML 檔完成的繁體中文二維太空飛行小遊戲。玩家從地球附近出發，駕駛一艘類似航海家號的探測船，依序靠近木星、土星、天王星、海王星與冥王星，學習如何利用行星重力改變航向與速度。

遊戲不是嚴格比例的天體力學模擬，而是把距離、時間、行星質量與速度都壓縮成適合遊玩的尺度。它的目標是讓玩家直覺理解「引力彈弓」如何讓太空船不用一直開引擎，也能在太陽系裡越飛越遠。

## 如何開始

直接用瀏覽器開啟：

```text
voyager_gravity_assist_mission_zh.html
```

不需要安裝套件、不需要後端伺服器，也不需要網路連線。

## 操作方法

- `W` 或 `↑`：推進。燃料有限，適合用來微調航向，不適合一路硬衝。
- `A` 或 `←`：向左轉。
- `D` 或 `→`：向右轉。
- `空白鍵`：暫停或繼續。
- 滑鼠滾輪：縮放鏡頭。
- 右側控制台的按鈕：可在觸控裝置或不想用鍵盤時操作。
- `時間倍率`：加快或放慢模擬時間。
- `鏡頭縮放`：拉近觀察飛船或拉遠看整體軌道。
- `飛行輔助`：選擇「自動朝向下一目標」時，船頭會自動轉向目前的任務航點。
- `重新開始`：重置整個任務。
- `跳到下一航點`：把飛船移到下一個目標附近，適合練習特定行星的掠飛。

任務航點依序是：

1. 木星
2. 土星
3. 天王星
4. 海王星
5. 冥王星

靠近目標行星到一定距離後，該航點會自動完成。撞上行星則任務失敗。

## 遊玩技巧

最重要的觀念是：推進器只是修正工具，真正的大幅轉向與加速要交給行星。

- 不要朝行星中心直直飛去，否則很容易撞上。
- 讓飛船從行星旁邊掠過，保留安全的最近距離。
- 若想加速前往外行星，通常要從行星公轉方向的後方掠過，讓行星像移動中的重力「把手」一樣把飛船帶出去。
- 若從行星前方掠過，飛船可能會被減速，這在真實任務中也有用途，例如降低近日點或調整軌道能量。
- 接近行星前先把船頭對準離開方向，接近後少量推進即可修正軌跡。
- 如果錯過目標，不一定要立刻重開；可以繞太陽一圈，等待新的交會機會。

## 引力彈弓的原理

引力彈弓也稱為重力助推、重力輔助或 swing-by。它利用三個物體之間的關係：太陽、正在繞太陽運動的行星，以及飛越行星附近的太空船。

在行星自己的參考系裡，太空船接近行星、被行星重力拉彎軌跡、再離開行星。若忽略大氣阻力與推進器，太空船離開行星時相對於行星的速度大小大致與接近時相同，只是方向改變了。

關鍵在於：行星本身也正以很高速度繞太陽移動。換回太陽參考系時，太空船的速度向量會與行星的公轉速度重新相加。只要幾何角度安排得好，太空船就能從行星龐大的公轉動量中拿到極小極小的一部分，表現為速度增加與軌道能量上升。反過來，如果從另一側掠過，太空船也可以把能量交還給行星，表現為減速。

行星因為質量巨大，失去或得到的動量比例微乎其微，幾乎無法測出；但對質量很小的太空船來說，這一點點交換就足以讓它飛往下一顆行星，或進入原本火箭無法直接達成的軌道。

這個遊戲把這件事簡化成幾個可見效果：

- 行星與太陽都會拉扯飛船。
- 飛船接近行星時，軌跡會被彎曲。
- 從不同方向掠過行星，會得到不同的離開方向與速度。
- 燃料很少，迫使玩家依靠重力而不是一直推進。
- 外行星路線模仿航海家任務的「大巡航」概念，但加入了冥王星作為遊戲終點。

## Star Trek 中的引力彈弓例子

《星際爭霸戰》系列多次提到「slingshot」或「gravitational slingshot」。和本遊戲最接近的畢凱艦長例子，出現在《銀河飛龍》（Star Trek: The Next Generation）第三季第六集〈Booby Trap〉。

在這一集中，企業號-D 進入一片古老戰場留下的小行星場，發現一艘普羅梅利安戰鬥巡洋艦，隨後企業號也陷入同一個能量陷阱。陷阱會吸收艦上的能量並把危險輻射反射回來，使企業號越來越難靠引擎脫困。

喬迪和畢凱最後選擇降低艦上能量輸出，改由人工操艦，而不是完全交給電腦。畢凱艦長親自接手舵位，利用小行星場中的重力環境與人工判斷，讓企業號以近似引力彈弓的方式繞過大型小行星，從設陷阱的小行星場中脫離。

嚴格說，這段劇情不是「脫離星球」，而是「逃離被設陷阱的小行星場」。但它很適合拿來理解本遊戲的核心感覺：駕駛者不是只靠引擎蠻力前進，而是抓住重力場、速度方向與時機，把一個危險的天體環境變成逃生路線。

## 真實使用過引力彈弓的太空任務

以下是現實中使用過重力輔助的著名任務。這裡的「衛星」泛指太空探測器、太空船與行星際任務。

| 任務 | 使用方式 | 重點 |
| --- | --- | --- |
| Mariner 10 | 飛掠金星後前往水星 | NASA 認為這是早期成功展示重力輔助的任務之一，讓探測器能多次接近水星。 |
| Voyager 1 | 利用木星重力前往土星 | 航海家 1 號在木星飛掠後被送往土星與泰坦方向。 |
| Voyager 2 | 木星、土星、天王星連續助推，前往海王星 | 經典「大巡航」路線，利用外行星排列把前往海王星的旅程從約 30 年縮短到約 12 年。 |
| Galileo | 以金星與地球重力助推前往木星 | Galileo 走 VEEGA 路線，透過 Venus-Earth-Earth Gravity Assist 逐步取得前往木星所需能量。 |
| Cassini-Huygens | 金星、金星、地球、木星助推後前往土星 | Cassini 透過多次重力輔助節省大量燃料，最後抵達土星系統。 |
| MESSENGER | 地球、金星與多次水星飛掠 | 這些重力輔助主要用來降低相對太陽的軌道能量，讓探測器能進入水星軌道。 |
| Rosetta | 地球與火星重力輔助 | ESA 的彗星任務用多次行星飛掠取得追上 67P/Churyumov-Gerasimenko 彗星的軌道。 |
| New Horizons | 木星助推前往冥王星 | 2007 年木星飛掠讓 New Horizons 更快抵達冥王星，並縮短航行時間。 |
| Juno | 2013 年地球飛掠助推前往木星 | Juno 先繞回地球附近，借地球重力取得前往木星所需速度。 |
| Parker Solar Probe | 多次金星重力輔助 | Parker 不是為了飛得更遠，而是用金星飛掠逐步降低近日點，讓探測器越來越接近太陽。 |
